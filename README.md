# Knockout

This is a POC Unity application loosely based as 3D version of Mike Tyson's Punchout with a political slant. A simple application that so far has only one character model that allows two players to fight each other using a variety of punches. The roadmap for development includes soundbytes from politicos along with incorporating ai designs based off of this paper
* https://research.facebook.com/publications/control-strategies-for-physically-simulated-characters-performing-two-player-competitive-sports/

## Features

* Two-player fighting game
* Variety of punches
* Simple controls
* In-game tutorial

## Requirements

* Unity 2019.4 or higher
* Visual Studio 2019 or higher
* Windows 7 or higher

## Getting Started

To get started, clone the repository and open the project in Unity. The project contains a scene with two characters and a boxing ring. To play the game, simply click on the "Play" button in the Unity editor.

## Controls
### *Work in Progress*
The controls for the game are as follows:

* Left mouse button: Punch
* Right mouse button: Block
* A: Move left
* D: Move right
* W: Move forward
* S: Move backward
